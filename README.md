# 善见童子

> 一个基于微信小程序的「圣杯」同人互动玩法

## 项目简介

「善见童子」灵感来源于芒果TV《大侦探》系列中的《焰火镇迷案2》"善见童子"部。这是一个模拟潮汕地区"圣杯"祈福文化的互动小程序，用户可以在令牌上写下问题，通过投掷糖饼获得童子的指示。

## 玩法说明

1. **提问**：点击「提问」按钮进入书写界面
2. **书写**：在令牌上写下想要询问的问题
3. **投掷**：点击「提问」将令牌和糖饼投入童子口中
4. **显灵**：等待童子显灵，获取批示结果
5. **解读**：根据糖饼结果（红红/红黑/黑黑）获得答案

### 结果解读

- **红红**：表示「是」
- **红黑**：表示「不确定」
- **黑黑**：表示「否」

## 技术栈

- 微信小程序原生开发
- Canvas 手写输入
- 自定义字体加载
- CSS3 动画效果

## 如何运行

1. 克隆项目到本地
   ```bash
   git clone https://github.com/MiseHinoha/ShanJianTongZi.git
   ```

2. 使用微信开发者工具打开项目文件夹

3. 填写你的 AppID（或选择测试号）

4. 点击「编译」即可预览
